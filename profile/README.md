# AnkleBreaker Studio

Small indie game studio passionate about games and their development. We build open-source tools that make game development bigger and more accessible.

## 🚀 Our Flagship Project: Unity MCP

**145+ tools** that give AI assistants like Claude full control over Unity Editor and Unity Hub.

| Repo | Description |
|------|-------------|
| [unity-mcp-server](https://github.com/AnkleBreaker-Studio/unity-mcp-server) | Node.js MCP server connecting AI assistants to Unity |
| [unity-mcp-plugin](https://github.com/AnkleBreaker-Studio/unity-mcp-plugin) | Unity Editor plugin running the HTTP bridge |

Scenes, GameObjects, components, scripts, builds, animation, physics, lighting, audio, profiling, Shader Graph — all controllable by AI.
