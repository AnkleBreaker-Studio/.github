## What We Do

We're a small studio based in France building two things we love: **games** and **developer tools**. Our MCP (Model Context Protocol) servers let AI assistants like Claude work directly inside your favorite dev tools — Unity, Discord, Jira, Plastic SCM, and more.

## Open Source

| Project | Description |
|---------|-------------|
| [**unity-mcp**](https://github.com/AnkleBreaker-Studio/unity-mcp-server) | 🎮 **145+ tools** — Full AI control over Unity Editor & Unity Hub. Scenes, GameObjects, scripts, builds, animation, physics, lighting, Shader Graph, and more. |
| [unity-mcp-plugin](https://github.com/AnkleBreaker-Studio/unity-mcp-plugin) | 🔌 Unity Editor plugin (UPM) — the HTTP bridge that connects your editor to AI |
| [AnkleBreaker-Core](https://github.com/AnkleBreaker-Studio/AnkleBreaker-Core) | 🧩 Core Unity scripts — no dependencies, ready to use |
| [AnkleBreaker-Utils](https://github.com/AnkleBreaker-Studio/AnkleBreaker-Utils) | 🛠️ Utility scripts for game development |
