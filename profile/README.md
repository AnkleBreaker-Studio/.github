## What We Do

We're a small studio based in France building two things we love: **games** and **AI-powered tools**.

Our MCP (Model Context Protocol) servers let AI assistants like Claude work directly inside the software you use every day — whether you're a game developer, a project manager, a video editor, or anything in between.

---

## 🎮 Unity MCP — Our Flagship Open-Source Project

| Project | Description |
|---------|-------------|
| [**unity-mcp-server**](https://github.com/AnkleBreaker-Studio/unity-mcp-server) | 🎮 **145+ tools** — Full AI control over Unity Editor & Unity Hub. Scenes, GameObjects, scripts, builds, animation, physics, lighting, Shader Graph, and more. |
| [unity-mcp-plugin](https://github.com/AnkleBreaker-Studio/unity-mcp-plugin) | 🔌 Unity Editor plugin (UPM) — the HTTP bridge that connects your editor to AI |

> **Unity MCP is and will always be free and open source.**
